# Userscripts Pack

## Script inclusi

- `Perplexity Search.user.js`
  - Redirect ricerca verso Perplexity (`?` => Grok, `!` => no redirect)
- `Search Results Cleaner.user.js`
  - Nasconde moduli rumorosi/sponsored sui principali motori
- `Search Query Shortcuts.user.js`
  - Prefissi rapidi: `yt`, `gh`, `so`, `npm`, `r`, `w`
  - Esempio: `gh react server components`
- `Copy Title and URL as Markdown.user.js`
  - Scorciatoia copia: `Cmd+Shift+C` (Mac) o `Alt+Shift+C`
- `Strip Tracking Parameters.user.js`
  - Rimuove `utm_*`, `gclid`, `fbclid` e altri parametri noti
- `Focus Mode Toggle.user.js`
  - Toggle focus: `Cmd+Shift+F` (Mac) o `Alt+Shift+F`

## Note Safari

- Verifica che l'estensione userscript sia autorizzata su tutti i siti.
- Se due script fanno redirect nello stesso contesto, lascia attivo solo quello desiderato per quel flusso.
